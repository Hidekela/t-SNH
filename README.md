# t-SNH


Ce projet qui a été fait en Binôme contient le code en **Python** de l'algorithme **t-SNE(tSNH)** qu'on a créé,le code de comparaison entre l'algorithme  original dans scikit learn et notre implementation ainsi que le rapport contenant les theories mathematiques derriere T-SNE.

## 🚀 Présentation

Le t-SNE est une technique de réduction de dimensionnalité non linéaire particulièrement efficace pour la visualisation de jeux de données complexes. 
### Fonctionnalités :
* Calcul des affinités Gaussiennes en haute dimension.
* Gestion de la **Perplexité** (recherche binaire des $\sigma_i$).
* Utilisation de la distribution **t-Student** pour l'espace de basse dimension.
* Optimisation par descente de gradient.
